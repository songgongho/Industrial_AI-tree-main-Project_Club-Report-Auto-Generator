# Industrial_AI-tree-main-Project_Club-Report-Auto-Generator
# 🏋️ 헬스동호회 정산 및 결과보고서 자동 생성기

## 📌 프로젝트 개요
* 네이버폼 엑셀 데이터를 기반으로 동호회비(자체비/보조비)를 정산하고, A4 규격의 결과보고서를 자동 생성하는 단일 HTML 웹 애플리케이션
* 매월 발생하는 복잡한 수기 정산 과정과 문서 작업 시간을 단축하고 업무 효율성을 극대화하기 위해 개발됨

## ✨ 주요 기능
* **엑셀 데이터 파싱 및 자동 정산** : 텍스트 복사/붙여넣기만으로 참여자별 물품 구매비와 헬스장 등록비를 자동 합산
* **스마트 공동구매 분배 로직** : 보조비 기준액(30,000원)을 채우기 위해 부족분 발생 시, 1,000원 단위로 올림하여 타인의 물품 구매비에서 자동으로 금액을 이동시키는 알고리즘 적용
* **인터랙티브 UI** : 자동 계산된 공동구매(지원금 이동) 내역을 표 형태로 제공하여 사용자가 직접 체크박스로 적용 여부를 제어하고 금액을 수정할 수 있음
* **직관적인 증빙 자료 첨부** : 화면 내 첨부 영역을 클릭하여 파일 탐색기로 이미지를 넣거나, 클립보드(Ctrl+V) 복사/붙여넣기를 통해 영수증 및 활동 사진을 손쉽게 삽입 가능
* **A4 인쇄 레이아웃 최적화** : 프린트 전용 CSS(`@media print`)를 적용하여 페이지 중간에 표나 이미지가 잘리는 현상을 방지하고, 마지막 장 지원금 신청서가 A4 1장에 완벽하게 들어맞도록 설계됨

## 🚀 사용 방법
1. 저장소의 `index.html` 파일을 다운로드하여 크롬, 엣지 등 웹 브라우저에서 실행
2. 화면 상단의 **1단계: 데이터 입력** 섹션에 활동 월, 총원, 대표자, 지원금 수령 계좌 등 기본 정보 입력
3. 네이버폼 응답 엑셀 텍스트를 복사(헤더 행 제외)하여 텍스트 영역에 붙여넣기
4. **[데이터 분석 및 공동구매 자동 설정]** 버튼을 클릭하여 할당된 내역 확인
5. **2단계**에서 공동구매 세부 내역을 확인 및 조율한 뒤 **[최종 보고서 렌더링]** 버튼 클릭
6. 화면에 렌더링된 보고서를 확인하고, 빈 사진/영수증 영역을 클릭하거나 이미지를 붙여넣어 증빙 자료 삽입
7. 우측 상단의 **[🖨️ 인쇄 / PDF 저장]** 버튼을 클릭하여 완성된 A4 문서를 출력 및 저장

* CSS3 (Flexbox, CSS Grid, Print Media Queries)
* Vanilla JavaScript (ES6+)
